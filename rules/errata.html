Clarification Reinforcement Wave Arrival Options
Added: April 2026

Question: How do units from Reinforcement Waves arrive on the battlefield?

Answer: When a Reinforcement Wave arrives at the start of a Battle Round, each unit in that wave can arrive using one of the following methods:

Method Requirements Placement
Battlefield Edge Any unit Wholly within 6" of your battlefield edge and more than 9" from enemy models
Deep Strike Unit must have Deep Strike ability As per the unit's Deep Strike rules (typically more than 9" from enemy models)
Surgical Deep Strike Unit must have Deep Strike ability + 1CP Stratagem As Deep Strike, but only needs to be more than 6" from enemy models
Surgical Deep Strike Limit: Only one unit per Reinforcement Wave can use this Stratagem.

Source: Wahapedia — Tactical Reserves
