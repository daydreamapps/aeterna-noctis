# Aeterna Noctis Campaign Rules Reference

## Battle Structure

### The Hulk (Nachmund)
- 1000-2000pt battles on the *Aeterna Noctis*
- Tracks Campaign Victory Points (CVP) and Strategic Site control
- Uses Nachmund Gauntlet missions

### The Interdiction (Maelstrom)
- 500-1000pt raids using *Lair of the Tyrant* rules
- Used to reduce enemy planetary control
- Can steal RP on threshold breaks

### The Breach (Phase 0)
- 500pt skirmishes for Intel Points

## Planetary Control

### Thresholds
| Control % | Uses Available |
|-----------|----------------|
| <30%      | 0 uses         |
| 30-59%    | 1 use per phase|
| 60%+      | 2 uses per phase|

### Rules
- Tracked via Administratum
- Wins increase control, losses decrease it
- No passive decay
- Uses calculated at phase start, no rollover

### Planet Boons
| Planet | Boon |
|--------|------|
| Ocularis Prime | Swap one Agenda after seeing opponent's choice |
| Ferrum IX | Remove one Battle Scar for 0 RP |
| Veridian Reach | +1 to Out of Action tests for one battle |
| Sanctum Malakor | +1 Starting CP for one battle |
| Aethelgard | Free "Fresh Recruits" Requisition |
| Void-Spire 7 | +1 SAP from your next Hulk victory |

### Raid Rewards
If a raid drops defender below a threshold (60% or 30%), raider gains +1 RP.

## Campaign Constraints

### The Sun Timer
As phases progress, apply Environmental Hazards (Solar Flare, Warp Instability) to narrative.

### The Betrayal Clause
Abandoning the Hulk for a Planet Raid = auto-Loss in Nachmund track.

## VOX-INTERCEPT Guidelines
- Must be under 2000 characters
- Use gothic 40K tone
- Reference unit crusade names
- Include thematic quote in footer

## M42 Timestamps
- Check most recent completed battle for current date
- Increment by 1 for each new battle
- Format: `[Number].M42` (e.g., 819.M42)
